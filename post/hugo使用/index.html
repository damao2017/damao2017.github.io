hugo使用

2019-28-28 20:50:50
hugo / 使用

hugo可以将md文件变成html页面，可以方便的发布到外网服务器上

  1. 下载hugo可执行文件

    我是windows平台，直接点击发布页，下载windows版本 https://github.com/gohugoio/hugo/releases

  2. 把hugo.exe文件所在的路径加入到环境变量path中

  3. 打开cmd，执行

    hugo new site notes

    将会创建一个notes文件夹，下面有一些文件。

  4. 文件夹内容如下

      archetypes/   --default.md 原始的md模板文件
  content/      --md文件存放处
  layouts/
  static/
  themes/       --主题文件
  config.toml   --配置文件

  5. default.md模板修改如下：

    +++ 
#默认此处应该是---,markdown显示到网页有问题，改成+++
title: "{{ replace .Name "-" " " | title }}"
date: {{ .Date }}
draft: false
tags: [""]
++++

  6. 下载主题文件

    官方主题地址 https://themes.gohugo.io/

    选择你喜欢的主题，去github，下载全部项目zip，然后复制到themes文件夹。

    我选的是这个https://github.com/yoshiharuyamashita/blackburn/archive/master.zip

    最终目录为/themes/blackburn

    archetypes/  
exampleSite/  
images/  
layouts/   
static/  ---/img/favicon.ico 网页图标
theme.toml
README.md

  7. 配置notes目录下的config.toml文件,这个文件需要结合主题文件的README.md 来写

    baseURL = "https://damao2017.github.io/"
LanguageCode = "zh-CN"
title = "damao's blog"
theme = "blackburn"
author = "damao2017"
copyright = "&copy; 9012. All rights reserved."
canonifyurls = true
paginate = 10
   
[indexes]
  tag = "tags"
  topic = "topics"
   
[params]
  BottomIntroduce = "damao's Blog"
  # Shown in the home page
  subtitle = "好好学习 天天向上"
  brand = "damao"
  #googleAnalytics = "Your Google Analytics   tracking ID"
  #disqus = "damao2017"
  # CSS name for highlight.js
  highlightjs = "androidstudio"
  dateFormat = "2006-02-02 15:04:04"
   
[menu]
  # Shown in the side menu.
  [[menu.main]]
    name = "Home"
    pre = "<i class='fa fa-home fa-fw'></i>"
    weight = 1
    identifier = "home"
    url = "/"
  [[menu.main]]
    name = "Posts"
    pre = "<i class='fa fa-list fa-fw'></i>"
    weight = 2
    identifier = "post"
    url = "/post/"
  [[menu.main]]
    name = "Tags"
    pre = "<i class='fa fa-tags fa-fw'></i>"
    weight = 3
    identifier = "tags"
    url = "/tags/"
  [[menu.main]]
    name = "About"
    pre = "<i class='fa fa-user fa-fw'></i>"
    weight = 4
    identifier = "about"
    url = "/about/"
     
   
 #[social]
  # Link your social networking accouns to the side menu
  # by entering your username or ID.
 #  weibo = "damao2017"
 #  twitter = "damao2017"
 #  github = "damao2017"

  8. 网站基本配置好了，现在开始写博客

    • 日常基本文章

      hugo new post/First.md

      生成到content/post目录下，刚刚已经配置了基础模板，所以生成的文件都有那个模板里面配置的东西。

      +++
#默认此处应该是---,markdown显示到网页有问题，改成+++
title: "First"
date: 2019-11-28T21:25:11+08:00
draft: false
tags: [""]
+++

      tags里面写上tag，draft是否为草稿，设置为false，后面会生成html。

    • about.md介绍类的，不经常修改

      hugo new post/about.md

      生成在content目录下，post平级，这里最后会被config.toml里的[[menu.main]]来引用。

      [[menu.main]]
    name = "About"
    pre = "<i class='fa fa-user fa-fw'></i>"
    weight = 4
    identifier = "about"
    url = "/about/"

  9. 启动本地调试

    hugo server --buildDrafts --watch

    –buildDrafts 代表草稿也编译，就是draft: true的

    –-watch或者-w 选项打开的话，将会监控到文章的改动从而自动去刷新浏览器，不需要自己手工去刷新浏览器

    打开本地地址http://localhost:1313/查看效果

  10. 生成本地的html页面

    在notes文件夹下，执行hugo命令即可，notes下的public文件夹会生成html网页。

  11. 发布到github pages

    git clone https://github.com/damao2017/damao2017.github.io.git
#删除damao2017.github.io.git文件夹下所有内容
git add .
git commit
#然后把.git文件夹移动到public文件夹
#在public文件夹下
git add .
git commit
git push
