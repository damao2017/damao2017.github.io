VUE 学习笔记

  1. vue cli 需要nodejs环境和webpack环境

  2. 安装vue cli

    npm install @vue/cli -g

  3. 查看版本

    vue --version

  4. 拉取cli 2.x模板

    npm install @vue/cli-init -g

  5. 2.x创建项目

    vue init webpack my-project-name

  6. 3创建项目

    vue create my-project-name

    runtime+compiler:template->ast->render->vdom->UI runtime+only: render->vdom->UI runtime+only 性能更高，需要的文件越小，以后主要用这个 main.js里面的写法不同 render : function(createElement){ //直接使用render将app模块覆盖index的div return createElement(app); }

  7. cli3 启动ui

    vue ui

    在创建里面选择加载项目，进入项目仪表盘，再操作项目

  8. 安装vuex

    npm install vuex --save

  9. 安装

    axiosnpm install axios --save

  10. 安装mockjs模拟接口数据， 用来拦截请求ajax请求并返回假数据

    npm install mockjs --save-dev

  11. 克隆项目

    git clone aaaa.git
npm install
